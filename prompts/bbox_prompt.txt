Given an image and a target-centric expression, return one tight bounding box around
the target object or target region.

Return strict JSON only:
{
  "box": [x1, y1, x2, y2],
  "confidence": 0.0,
  "reason": "brief visual grounding reason"
}

Rules:
- Coordinates must be pixel coordinates in the original image.
- Use [x1, y1, x2, y2], where x1/y1 are top-left and x2/y2 are bottom-right.
- If the expression refers to a part or region, box that part or region, not the whole object.
- Do not include markdown or explanations outside JSON.
